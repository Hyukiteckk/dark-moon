Personalização de Perfil

Gere códigos 3y3 para customizar seu perfil no Discord

Banner do Perfil

Cole o link direto de uma imagem — outros usuários com o plugin verão como seu banner.

✅ Funciona com qualquer host: Imgur, Catbox, Giphy, etc.
Use link direto terminando em .gif / .png / .jpg — não use links de galeria ou página.
Dica: botão direito na imagem → "Copiar endereço da imagem".

Cores do Perfil

Define um gradiente personalizado no fundo do seu perfil (visível para quem tem o plugin).

Como usar

  1. Gere o código acima e copie-o
  2. Abra o Discord → clique no seu avatar → Editar Perfil
  3. Cole o código invisível no campo Sobre mim (Bio)
  4. Salve — o código fica invisível, isso é normal
  5. Outros usuários com o plugin OrionQuests instalado verão seu banner/cores
Você pode combinar múltiplos códigos (banner + cores) colando ambos na mesma bio. Cada código ocupa espaço invisível.
